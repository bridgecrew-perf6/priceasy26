Brief Introduction :-


This is demo implementation for "PRICEASY-THE PRICE CONSULTANT " model which is making suggestions for cost of second handed cars. We are using only liner regression in this prototype model.We have used our 80% of data to train this model and 20% data for testing.As of initial bases we have used data model from "Kaggle.com" website.We have also examined test score of this model and it is "0.8626750969737311" , which means model is having 86% accuracy.
